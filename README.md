# 星星骰娘-重骰版！

为 Astrbot 设计的 TRPG 骰子插件（参考 [Dice!](https://forum.kokona.tech/) 与 [海豹](https://dice.weizaima.com/) 的实现），提供掷骰、技能判定、人物卡管理与日志导出等常用功能，便于在群聊中进行跑团玩法支持。

为所有使用astrbot平台但想要有与其他骰娘相同体验的骰主设计。

基于[TRPGdice-Complete](https://github.com/WhiteEurya/Astrbot_plugin_TRPGdice-Complete)二次开发，提供配置文件以管理骰娘输出风格。

---

## 安装

**推荐**：使用Astrbot插件市场安装

或：

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/HosisoraLing/astrbot_plugin_trpgdice_rerolled.git
   ```

2. 将插件文件/目录放入 Astrbot 的插件目录。

3. 启动或重启 Astrbot，确认控制台输出已加载本插件。

## 功能

- **基础掷骰**
  - 支持常见的 DnD / CoC / 跑团检定
  - 支持算式，例如 `1d100+5`、`3d6+2d4-1d8`
  - 支持保留最高骰，例如 `10d6k5`（掷10个d6保留最高5个）
  - 支持重复掷骰，例如 `3#1d20`（掷1d20三次）
  - 支持小众规则的掷骰 (吸血鬼规则)

- **角色卡与检定**
  - 支持角色属性绑定与管理
  - 支持 **技能检定** (普通/奖励骰/惩罚骰)
  - 支持 **理智检定** (SAN Check) 与疯狂症状生成
  - 支持 **技能成长** 判定

- **先攻系统**
  - 先攻掷骰与排序
  - 回合推进管理

- **日志与记录**
  - 自动保存跑团对话日志
  - 支持导出 JSON 格式记录，兼容主流染色器
  - 支持暂停/恢复/删除日志会话

- **自定义风格回复**
  - 所有输出文本均可在 Astrbot 插件配置中自由修改
  - 每个掷骰语句、判定结果、系统提示都支持个性化定制
  - 配置项说明中标注了可用的变量占位符

## 快速使用示例

> 插件加载后，可在群聊中使用下列示例指令进行测试（支持 `.` `。` `/` 作为前缀）

* 基本掷骰：
  ```
  .r 1d100          # 掷1个100面骰
  .r 3d6+5          # 掷3个6面骰+5
  .r 10d6k5         # 掷10个d6，保留最高5个
  .r 3#1d20         # 掷1d20三次
  ```

* 技能判定：
  ```
  .ra 侦查 50       # 进行侦查检定（技能值50）
  .rab 2 射击       # 带2个奖励骰的射击检定
  .rap 1 闪避       # 带1个惩罚骰的闪避检定
  ```

* 人物卡管理：
  ```
  .pc create <名字>   # 创建人物卡
  .pc show             # 显示当前人物卡
  .pc change <名字>    # 切换人物卡
  .st san-5            # 修改属性值
  ```

* 理智检定：
  ```
  .sc 1d6/1d10       # SAN Check（成功损失1d6/失败损失1d10）
  .ti                # 临时疯狂
  .li                # 长期疯狂
  ```

* 先攻系统：
  ```
  .ri                # 掷先攻
  .ri +3             # 掷先攻（+3调整值）
  .init              # 查看先攻列表
  .ed                # 结束当前回合
  ```

* 日志管理：
  ```
  .log new <日志名>   # 开始新日志
  .log off            # 暂停日志
  .log on             # 恢复日志
  .log end            # 结束并导出日志
  ```

如需完整指令说明，请运行 `.dicehelp` 查看。

## 配置

所有输出文本和参数均可在 Astrbot 的插件配置界面中修改，无需编辑代码。

配置分类：

- **output** - 所有输出文本模板（支持 `{变量名}` 占位符）
- **dice** - 掷骰参数（默认面数、最大个数/面数等）
- **initiative** - 先攻掷骰范围
- **sanity** - 理智检定和疯狂症状配置
- **character** - COC/DND 角色生成骰子公式
- **coc_rules** - 大成功/大失败规则
- **growth** - 技能成长阈值
